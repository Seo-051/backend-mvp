# backend-mvp

## MVP1 (1차 목표)

### 목표
프론트-백엔드-DB 연결을 완주하고, 배포까지 가능한 기본 게시판을 구현한다.
인증(JWT)과 권한(작성자만 수정/삭제), 페이징/검색 등 백엔드 기본기를 증명한다.

### 포함 기능
- 회원가입 / 로그인 (JWT)
- 내 정보 조회
- 게시글 CRUD
- 게시글 목록 페이징
- 게시글 검색(제목 or 제목+내용)
- 권한 체크(작성자만 수정/삭제)

### 제외 기능 (MVP2로)
- 댓글
- 파일 업로드
- 좋아요/조회수
- 관리자 기능
- 마크다운 에디터
